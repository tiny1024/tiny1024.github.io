本文介绍如何在Windows的电脑上制作Mac OS X的启动U盘。

通常我们制作Mac系统的安装盘，都是在Mac电脑上进行的。但如果遇到Mac系统不可用并且Recovery HD分区也不可用的话，我们就只好在Windows上制作启动盘了。本文介绍一个可在Windows电脑上制作Mac系统安装U盘的操作方法，从而可以借助Windows电脑来制作并恢复系统。

需要准备的工具

Windows电脑：Windows 7，Windows 8或者Windows 10都可以
U盘：容量至少8G，读写速度要快一些的，否则会很慢

第一步 准备软件

下载 TransMac

官方网址：http://www.acutesystems.com/scrtm.htm
下载安装之后有15天的试用期。

下载 OS X

从我的网盘下载：http://pan.baidu.com/s/1ntjFtXZ 密码: huiz
系统版本：OS X EI Capitan (15A284)

第二步 制作Mac系统安装U盘

  1. 把U盘接入电脑，打开TransMac软件，在左侧窗口找到U盘。
  2. 将U盘格式化为Mac下的格式：选中U盘右键选择”Format Disk for Mac”，选择默认的Mac OS Extended HFS+即可，点击OK进行格式化。
  3. 格式化完成后，选中U盘右键选择”Restore with Disk Image”选项，选择刚刚下载的OS X系统镜像文件(.dmg)。
  4. 等待软件将Mac系统的镜像写入U盘。

第三步 启动Mac安装OS X

  1. 接入刚刚制作的启动U盘。
  2. 启动Mac，并按住option键，系统显示可用启动盘后选择U盘并进入。
  3. 接下来选择Install OS X。
  4. 接下来泡杯茶，打开一本书，等着安装完成吧。
